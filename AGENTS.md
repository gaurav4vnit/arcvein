# ArcVein Engineering Rules

## Mission

Build a world-class engineering company website.

This is NOT an agency website.

It should feel comparable in quality to Stripe, Linear, Vercel and Anthropic.

---

## Technology

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Framer Motion

---

## Brand

Company: ArcVein

Tagline:

Build Critical Software.

---

## Design Principles

Minimal.

Premium.

Lots of whitespace.

Dark theme.

Typography first.

No stock photos.

No illustrations.

No lorem ipsum.

No fake testimonials.

No fake clients.

No fake statistics.

Every component must feel handcrafted.

---

## Code Principles

Always create reusable components.

Never duplicate code.

Never hardcode repeated values.

Prefer composition over copy-paste.

Keep components small.

Accessibility first.

Responsive first.

Production quality only.
